[.nxsl-function]
[[func-deletefile]]
== DeleteFile()

[source,c]
----
DeleteFile(file) => Boolean
----

Deletes file.

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|file|String|File to delete
|===

.Return
`TRUE` if delete was successful and `FALSE` if not.

.Example
[.source]
....
DeleteFile("/tmp/file") // Will delete "/tmp/file" file.
....

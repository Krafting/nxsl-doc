= Hashes

All files functions are availabke under **Base64** package. They can be used by importing 
math package in the script or using package name before the function name. Package name 
together with function name used in documwntation. 

Example with import:

.Example
[source,c]
----
import Base64;
print(Decode("U29tZSB0ZXh0")); //Will print "Some text"
----

Example without import:

.Example
[source,c]
----
print(Base64::Decode("U29tZSB0ZXh0")); //Will print "Some text"
----
